# Dusty Tennis Shop
Tennis Ecommerce for WDD 340 Web Backend Development Final Project

## 1. Project Description
Dusty Tennis Shop is a online marketplace and community for consumer tennis string needs. The system will allow users to purchase strings from a wide selection of strings, read our in-house tester/customer product reviews, and provide their own customer experience feedback on items.

## 3. User Roles
### Administrator
* Full access permissions
* User Account read/write permissions
* Review read/write permissions
* Product read/write permissions

### Product Tester (In-house):
* Product tester review section read/write permissions
* Review Response read/write permissions
* Purchase product at discounted rate
  
### User:
* Customer review section read/write permissions
* Product tester review section read permissions
* Purchase product at full price
